Estrutura Básica do HTML

A estrutura básica de um documento HTML consiste nestes elementos:

  • <.!DOCTYPE html>: Define o tipo de documento (HTML5).
  • <.html>: Elemento raiz que engloba todo o conteúdo.
  • <.head>: Contém metadados (configurações, links externos, título da página).
  • <.meta charset="UTF-8">: Define a codificação de caracteres.
  • <.title>: Define o título da aba do navegador.
  • <.link>: Importa folhas de estilo externas (CSS).
  • <.body>: Contém todo o conteúdo visível (textos, imagens, links, etc.).

TAGS

  • TAGS são elementos usados para marcar e estruturar o conteúdo de uma página web.
  • Cada tag possui uma abertura (<.tag>) e, geralmente, um fechamento (<./tag>), exceto as tags auto-fechadas (<.tag />).
  • As tags podem conter atributos que fornecem informações adicionais, como class, id, src, href, etc.
  • Função: Definir cabeçalhos, parágrafos, links, imagens, tabelas, formulários, entre outros elementos da página.

TEXTOS

Tags para formatação de textos:

  • <.h1 até <.h6>: Títulos e subtítulos (h1 é o maior, h6 o menor).
  • <.p>: Define parágrafos.
  • <.strong>: Deixa o texto em negrito.
  • <.i>: Deixa o texto em itálico.
  • <.br>: Quebra de linha.
  • <.hr>: Linha horizontal (divisor).
  • <.mark>: Marca o texto com destaque.
  • <.small>: Texto pequeno.
  • <.blockquote>: Citação de bloco.
  • <.span>: Marca um trecho do texto (inline) para aplicar estilos ou scripts.

Listas Ordenadas e Não Ordenadas

Você pode criar listas em HTML facilmente utilizando as tags.

  • Lista Ordenada (<.ol>): Exibe itens numerados.

    1. Primeiro item
    2. Segundo item
    3. Terceiro item

  • Lista Não Ordenada (<.ul>): Exibe itens com marcadores (pontos, círculos, etc.)

    • Item 1
    • Item 2
    • Item 3

Diferença:

  • Ordenada: Sequência numérica ou alfabética.
  • Não Ordenada: Apenas marcadores, sem ordem definida.

LINKS

Links (ou texto âncora) são importantes para ligar conteúdos em documentos HTML.

  • Criados com a tag <.a> (âncora).
  • O atributo principal é o href, que define o destino do link.

Atributos comuns:

  • href: URL ou caminho do destino.
  • target="_blank": Abre o link em uma nova aba.
  • title: Texto exibido ao passar o mouse sobre o link.

Função:

  • Permite navegar entre páginas, acessar documentos ou iniciar ações, como enviar e-mails.
